Overview

SaifOS is a browser based operating system created using HTML, CSS, and JavaScript. The project simulates a desktop environment inside a web page and allows users to interact with multiple draggable windows, similar to a real operating system.

This project was built for the WebOS 1 challenge and focuses on learning core web development concepts while creating a fun and interactive user experience.

Features
Multiple draggable windows
About application window
Notes application for writing text
Live digital clock
Desktop wallpaper changer
Dark mode toggle
Taskbar navigation system
No login or password required
Technologies Used
HTML5
CSS3
JavaScript
How I Built It

I started by creating the basic desktop layout using HTML and CSS. After building the desktop and taskbar, I added application windows for the About page, Notes app, and Clock app.

Next, I used JavaScript to make the windows draggable so users could move them around the screen. I then implemented a live clock that updates every second.

To make the project more unique, I added extra features that were not part of the basic requirements, including a wallpaper changer and a dark mode option. These features allow users to customize the appearance of the desktop.

Finally, I improved the styling and organization of the code to create a cleaner and more realistic operating system experience.

Challenges

One of the biggest challenges was implementing draggable windows. I had to learn how mouse events work in JavaScript and how to update the position of elements dynamically.

Another challenge was making sure all windows and controls worked together without interfering with each other.
